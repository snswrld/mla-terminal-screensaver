Sick of suspicious network connections when you walk away from your Win11 machine? Looking for malicious or surveilling IP address to add to your firewall rules? 
Like keeping abreast of the headlines while your newreader logs network connections that may be unwanted and largely impermissible?
This package uses wireshark and powershell and the old control panel interface for setting a screen saver so you don't have to stare at that Windows lock screen,
while also learning how many networks an unused computer is connecting to.
